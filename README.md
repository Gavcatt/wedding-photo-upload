# Wedding Photo Upload Landing Page

A mobile-first digital invitation page for wedding guests to upload photos privately to Dropbox.

## What it includes

- `index.html` — mobile-first invitation layout and QR-ready upload CTA
- `styles.css` — animated, responsive mobile styling
- `script.js` — private Dropbox link and QR code generation

## Setup

1. Create a Dropbox File Request and copy the request URL.
2. Replace `YOUR_DROPBOX_FILE_REQUEST_ID` in `script.js` with that URL.
3. Host the folder on any static site provider (GitHub Pages, Netlify, Vercel, etc.).
4. Share the deployed page URL as a QR code or digital invite link.

## Notes

- This page is optimized for phones and digital guest experiences.
- Guests can also scan the QR code to open the upload page from another device.

## Deployment

- GitHub Pages:
  1. Create a new GitHub repository and push these files.
  2. In the repo settings, enable GitHub Pages from the `main` branch and root folder.
  3. Open the provided site URL and share it with guests.
- Netlify/Vercel: connect the repo and deploy the folder as a static site.
